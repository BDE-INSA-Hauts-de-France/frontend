## La TODO List du nouvel insaïen !
- [ ] **Créer ton compte UPHF** : Tu recevras un email avec les instructions.
- [ ] **Télécharger l'application UPHF** : Retrouves-y ton emploi du temps, tes mails, notes et toutes les actus.
- [ ] **Rejoindre le Discord de l'INSA HdF** : Connecte-toi avec tes futurs camarades.
- [ ] **Abonne toi au Instagram du BDE et de l'INSA** : [@bdeinsahdf](https://instagram.com/bdeinsahdf) et [@insahdf](https://instagram.com/insahdf)
- [ ] **Préparer ton arrivée** : Consulte les guides sur le logement, les transports et le campus.
- [ ] **Participer à la semaine d'intégration** : Ne manque pas les événements de rentrée pour faire connaissance.
- [ ] **Découvrir les associations** : Explore les clubs et activités pour t'impliquer dès le début.
- [ ] **Faire connaissance avec les services de l'école** : Bibliothèque, santé, sport, etc.
